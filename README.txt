Download Docker: Download docker and open it.

Create the Docker container: Run docker compose up -d in the folder that contains the docker-compose.yml file.

Access your local host 5050: Open your web browser to localhost5050, this will bring you to a pgAdmin page

Create a server: Create a new server within pgAdmin that your database will run through. 

(Within your project folder) Run the node server: run node server.js to start the node server, this will create the necessary tables in the database and allow the site to run.

Open index.html: open index.html and use the BAC website!
